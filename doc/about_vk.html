Valkyrie

Valkyrie is a graphical user interface for Valgrind: www.valgrind.org.

This program was built using Qt v%1, and is running using Qt v%2.

You can download the latest version of Valkyrie from the OpenWorks website: http://www.open-works.co.uk/projects/valkyrie.html.

For more information on Valkyrie and/or Valgrind, contact info@open-works.co.uk.
